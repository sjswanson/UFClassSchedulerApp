All files used in this project are in the "Files" folder.

There is only one C++ source code file. It is called "main.cpp". It must be compiled with the C++11 standard. (i.e. -std=c++11)

In addition, there is one other file called "main.out", which is the compiled version (on Linux) of "main.cpp".

All other files are the HTML files used by "main.cpp" and "main.out" to access the different sections of courses that are requested by the user.
